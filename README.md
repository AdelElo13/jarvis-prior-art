## What Makes Engine 9 (Director) Unprecedented

The individual components exist elsewhere in isolation. But no one has combined them into a single multiplexer:

1. **Alien systems driving tool selection** — Everywhere else, tools are selected by keyword matching or popularity. Nowhere do cognitive systems (metabolism, intent entropy, domain trust) steer which tools are selected or suppressed.

2. **Synthesized meta-directive** — LLMs everywhere get loose context blocks. Nobody compresses 5 cognitive signals into a single instruction the LLM MUST follow.

3. **Cognitive veto on tools** — Security gates exist everywhere (firewalls, policy engines). But a veto based on *cognitive appropriateness* (domain trust too low, wrong metabolic state) — that does not exist.

4. **Mid-request coherence refresh** — Everywhere, cognitive state is computed once at the start. Nobody recomputes after tool failures.

5. **Full alien-to-background-agent pipeline** — Autonomous agents exist, but none execute background actions based on knowledge gaps, metabolic state, intent entropy, and domain trust combined.
